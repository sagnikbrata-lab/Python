# Multi Fermi Estimation Engine

A menu-driven Python application that explores classic Fermi estimation problems.

## Features

- Interactive menu system
- Adjustable assumptions and parameters
- Real-time recalculation of estimates
- Multiple estimation modules

## Included Estimators

### 1. Chicago Piano Tuners Problem
Estimate the number of professional piano tuners required in Chicago (original Fermi-Estimation problem).

### 2. West Bengal Harmonium Tuners Problem
Estimate the number of professional harmonium tuners required across West Bengal.

### 3. World's Text Stack Problem
Estimate the height of a stack of paper containing every text message sent in a year and compare it to the Earth-Moon distance.
